    content: |
      GroupDocs.Assembly untuk .NET API membantu Anda membangun otomatisasi dokumen yang kuat dan aplikasi pembuatan laporan dengan kemampuan untuk menghasilkan laporan dari templat yang disesuaikan di C#, ASP.NET, dan aplikasi terkait .NET lainnya. Hanya dengan beberapa baris kode, pustaka pelaporan .NET dengan cerdas mengumpulkan data yang diberikan dari templat dokumen yang ditentukan dan menghasilkan laporan yang indah dalam format keluaran yang disukai dengan mengambil data dari berbagai sumber data (basis data, XML, JSON, ODATA, CSV, Objek .NET kustom).  

      Ini mendukung sintaks template berbasis LINQ, dan pengguna dapat dengan mudah menghasilkan dokumen keluaran dalam semua format file bisnis yang umum digunakan seperti PDF, HTML, email Outlook, Microsoft Office Word, lembar kerja Excel, presentasi PowerPoint dan slide. Properti pemformatan untuk elemen templat juga dapat dikonfigurasi dengan memanipulasi teks, HTML & blok bersyarat, gambar, bagan, kode batang, hyperlink, tabel pivot, dll.  

      GroupDocs.Assembly untuk .NET dapat digunakan untuk mengembangkan aplikasi di lingkungan pengembangan apa pun yang menargetkan platform .NET. Ini kompatibel dengan semua bahasa berbasis .NET dan mendukung sistem operasi populer (Windows, Linux, MacOS) di mana kerangka kerja Mono atau .NET (termasuk .NET Core) dapat diinstal.

      - title: "Integrasi Mudah"
        content: |
          Anda dapat dengan mudah mengintegrasikan GroupDocs.Assembly untuk .NET API dengan aplikasi .NET Anda hanya dengan menggunakan beberapa baris kode. Berikut adalah contoh kode untuk menghasilkan laporan dalam format dokumen terbuka:

          ```cs
          // Membuat instance kelas DocumentAssembler
          DocumentAssembler assembler = new DocumentAssembler();
          //Panggil AssembleDocument untuk menghasilkan laporan
          assembler.AssembleDocument("D:\\WordTemplates\\Barcode.docx", "D:\\WordReports\\Barcode.docx", new DataSourceInfo(DataLayer.GetCustomerData(), 
          "customer"));
          //(Lihat detail metode DataLayer.GetCustomerData() di https://github.com/groupdocs-assembly/GroupDocs.Assembly-for-.NET)
          ```
